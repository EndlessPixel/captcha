验证码API使用示例

1. 获取验证码

调用 /api/captcha 接口获取验证码图片和token

GET /api/captcha

2. 验证验证码

使用获取到的token和用户输入的验证码进行验证

POST /api/verify

3. 查询验证状态

使用验证成功后获取的uid查询验证状态

GET /api/inquire?uid={uid}

API使用说明

  • 获取验证码：调用 GET /api/captcha，返回包含token和base64编码的验证码图片
  • 验证验证码：调用 POST /api/verify，提交token和用户输入的验证码
  • 查询验证状态：调用 GET /api/inquire?uid={uid}，查询验证是否成功
  • 验证码有效期为5分钟，验证成功后会生成一个24小时有效的UID
  • API接口支持跨域请求，可在任何网站中调用
