验证码SDK使用示例

基本用法

只需在页面中创建一个div元素，并引入验证码SDK，验证码会自动加载。

<!-- 在页面中添加一个div容器 --> <div id="captcha-container"></div> <!-- 引入验证码SDK --> <script src="/captcha-sdk.js"></script>

高级用法

可以通过配置选项自定义验证码的行为和回调函数。

// 自定义配置 window.initCaptcha({ containerId: 'custom-captcha', onSuccess: function(uid) { console.log('验证成功，UID:', uid); document.getElementById('custom-result').textContent = '验证成功，UID: ' + uid; }, onError: function(message) { console.log('验证失败:', message); document.getElementById('custom-result').textContent = '验证失败: ' + message; }, onExpire: function() { console.log('验证码已过期'); document.getElementById('custom-result').textContent = '验证码已过期'; } });

SDK功能特点

  • 自动加载验证码
  • 支持点击图片刷新验证码
  • 实时验证反馈
  • 自定义回调函数
  • 响应式设计
  • 无依赖，纯原生JavaScript
